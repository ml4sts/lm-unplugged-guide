# LLMs UnPlugged Facilitator Guide

This site is a guide for facilitators on how to run the `LLMs Unplugged` Guide developed by the [URI ML4STS Lab](ml4sts.com). This guide is developed in collaboration with [Kenya Andrews](). 

This guide is intended for reference use after attending a workshop. If you would like to attend a faciliator workshop, [sign up]().
